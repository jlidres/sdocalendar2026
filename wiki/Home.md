## Project Summary

This repository contains a Next.js 16 app for viewing and managing the DepEd SDO San Juan City school-year calendar for SY 2026–2027.

Key capabilities:

- Public calendar dashboard (`/`)
- Admin page for managing activities (`/admin`)
- PostgreSQL persistence through Prisma
- Calendar/event API (`/api/events`)

## Core Stack

- Next.js 16 (App Router)
- React 19
- Prisma ORM
- PostgreSQL
- TypeScript
